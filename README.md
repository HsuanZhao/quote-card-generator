# 📜 名言卡片工作室（Quote Card Studio）

> 一款可直接运行的 HTML 工具，用于生成具有 AI 科技 / 法律双风格的名言卡片，并一键导出高清 PNG。

## ✨ 功能特点

- **实时编辑**：卡片上的引用、作者、身份、编号均可直接在页面中点击修改，同步更新表单。
- **双圈层切换**：一键在「AI 科技」与「法律」之间切换，自动填充对应的示例内容与徽标。
- **四种主题**：
  - 暗夜矩阵（AI）
  - 极简白皮书（AI）
  - 卷宗黑金（法律）
  - 象牙判例（法律）
- **徽标显隐**：可自由控制卡片右下角的装饰徽标。
- **高清导出**：借助 `html2canvas`，以 3.33 倍缩放导出约 1500×2000 像素的 PNG 图片，适合社交媒体分享（公众号、小红书、朋友圈等）。

## 🚀 快速开始

1. 下载本仓库的 `quote-card-generator.html` 文件（或直接打开在线 Demo）。
2. 双击该文件，在浏览器中打开。
3. 在左侧面板中：
   - 修改引用文字、作者、身份、编号；
   - 切换圈层与主题；
   - 点击「下载高清卡片 · PNG」即可导出。

> 无需安装任何依赖，所有功能在浏览器本地运行。

## 🛠️ 技术栈

- 原生 HTML + CSS + JavaScript
- [html2canvas](https://html2canvas.hertzen.com/) – 用于将 DOM 转换为 Canvas 并导出图片

## 📝 自定义提示

- 如需添加新的主题，请在 CSS 中定义 `.card.theme-xxx` 的样式，并在 JavaScript 中为对应的 `.theme-swatch` 绑定 `data-theme` 属性。
- 若要修改导出图片尺寸，可调整 `html2canvas` 调用中的 `scale` 参数（当前为 3.33）。
